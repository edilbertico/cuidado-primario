# APS Cundinamarca — Sistema Inteligente de Evaluación APS

Plataforma web institucional para la autoevaluación de la estrategia de Atención Primaria en Salud (APS), basada en el **Informe Mundial sobre APS de la OMS (2024)** y la **Declaración de Astaná (2018)**.

## 📋 Características

- **14 Regiones de Salud** de Cundinamarca
- **116 Municipios** con selección jerárquica obligatoria
- **122 Hospitales/ESE** participantes
- **8 Palancas Operativas** del Marco OMS/UNICEF 2024
- **40 Preguntas de autoevaluación** con escala 1-5
- **3 Esferas de la APS** (Servicios Integrados, Participación Comunitaria, Acción Multisectorial)
- **4 C de Starfield** (Primer contacto, Continua, Integral, Coordinada)

## 🎯 Módulos

### Usuario Evaluador
- 🎓 Bienvenida educativa obligatoria (3 Esferas + Astaná 2018 + Glosario)
- 📝 Autoevaluación con barra de progreso y guardar avances
- 📊 Resultados inmediatos al enviar (score, semaforización, plan de acción)
- 🌍 Casos de País (20 países con buenas prácticas)
- 📖 Glosario interactivo (30 términos con definición, concepto, ejemplo, normatividad)

### Super Administrador
- 📊 Dashboard Ejecutivo consolidado con filtros múltiples
- 👥 Gestión de usuarios (aprobación, activación, bloqueo)
- 📋 Registros con trazabilidad completa
- 🏆 Ranking de hospitales
- 🎯 Plan de Acción inteligente (auto-generado + manual, 16+ campos)
- ⏱️ Seguimiento con cronogramas automáticos y alertas
- ⚙️ Configuración con recuperación de usuarios y contraseñas
